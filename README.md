# 🎧 G-Spotify — Futuristic Music Experience
G-Spotify is a modern Google × Spotify inspired web application designed with a premium glassmorphism interface, neon aesthetics, and immersive UI interactions.
Built using HTML5 and CSS3, the project delivers a sleek music streaming experience with responsive layouts, smart search functionality, and futuristic visual effects.

## ✨ Highlights

* Minimal & premium UI design
* Google-inspired smart search interface
* Spotify-style music dashboard
* Advanced CSS animations & hover effects
* Glassmorphism + neon glow aesthetics
* Fully responsive modern layout
* Interactive music player section

## 🛠️ Technologies Used

* HTML5
* CSS3
* Responsive Web Design
* Flexbox & CSS Grid
* Glassmorphism UI Design
* CSS Animations & Transitions
